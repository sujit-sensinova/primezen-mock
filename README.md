# Primezen | Luxury Smart Home Automation

Primezen is an elite, high-performance landing page for a premier smart home automation brand in India. Built with a "Light Luxury" aesthetic, it prioritizes minimalism, sensory feedback, and cinematic movement.

## ✨ Key Features

- **Level 1-3: Light Luxury Foundation**: Pure white aesthetic, editorial typography (GC Epic Pro & Gilroy), and a full-bleed cinematic hero.
- **Level 4-5: Sensory & Movement**: Film grain overlays, magnetic interactions, custom Web Audio API click sounds, and a 3D horizontal scrolling gallery.
- **Level 6: Production Grade**: Optimized for SEO (OG tags, JSON-LD), performance (lazy loading, video posters), and high-quality hardware imagery.
- **Level 7: Apple-Tier UX**: Context-aware smart cursor that morphs based on hover (DRAG, VIEW, TOGGLE) and kinetic typography that reacts to scroll velocity.
- **Level 8: Deep Immersion**: 
  - **Global Theme Journey**: Background smoothly transitions from Light to Midnight Stealth (dark mode) as you scroll through the portfolio.
  - **Liquid Magnetic Text**: Individual letters in the CTA physically pull towards the cursor.
  - **High-Tension Preloader**: Cinematic 0-100% counter with custom easing.

## 🚀 Tech Stack

- **Framework**: React 18 + Vite
- **Styling**: Tailwind CSS + Vanilla CSS
- **Animation**: Framer Motion
- **Smooth Scroll**: Lenis
- **Sound**: Web Audio API
- **Icons**: Lucide React

## 🛠️ Getting Started

1. **Clone the repo**:
   ```bash
   git clone https://github.com/sujit-sensinova/primezen-mock.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
